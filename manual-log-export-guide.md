# 手動ログエクスポート手順

## 📋 開発者ツールでのログ収集手順

### 1. **Consoleタブのログ収集**

#### 基本的な手順
1. **VS Code/Cursorを開く**
2. **開発者ツール** (F12) を開く
3. **Consoleタブ** を選択
4. **Clear** ボタンでログをクリア
5. **拡張機能で質問を送信**
6. **エラーメッセージを確認**

#### エラーのみを表示
1. Consoleタブの **Filter** アイコンをクリック
2. **Errors** にチェックを入れる
3. エラーメッセージのみが表示される

#### ログのコピー
1. **Ctrl+A** で全選択
2. **Ctrl+C** でコピー
3. テキストエディタに貼り付け

### 2. **Networkタブのリクエスト確認**

#### リクエストの確認
1. **Networkタブ** を選択
2. **Filter入力欄** に `supabase` と入力
3. **拡張機能で質問を送信**
4. リクエストが表示されるか確認

#### リクエスト詳細の確認
1. リクエストをクリック
2. **Headers** タブでリクエストヘッダーを確認
3. **Response** タブでレスポンス内容を確認
4. **Preview** タブでレスポンスの構造を確認

### 3. **各拡張機能の特定ログ**

#### Continue拡張機能
- **Command Palette** (Ctrl+Shift+P)
- `Continue: Show Logs` を実行

#### CodeGPT拡張機能
- **Command Palette** (Ctrl+Shift+P)
- `CodeGPT: Show Logs` を実行

#### Cline拡張機能
- **Command Palette** (Ctrl+Shift+P)
- `Cline: Show Logs` を実行

### 4. **ログの整理と分析**

#### 収集すべき情報
- **エラーメッセージ**: 具体的なエラー内容
- **リクエストURL**: 実際に送信されたURL
- **レスポンスステータス**: HTTPステータスコード
- **レスポンス内容**: 返されたJSONデータ
- **タイムスタンプ**: エラーが発生した時刻

#### 分析のポイント
- **リクエストが送信されているか**
- **レスポンスが返ってきているか**
- **レスポンスの形式が正しいか**
- **拡張機能がレスポンスを解析できているか**

## 🔧 トラブルシューティング

### よくある問題
1. **リクエストが送信されていない**
   - 拡張機能の設定が正しくない
   - APIエンドポイントが間違っている

2. **リクエストは送信されるがレスポンスがない**
   - API側でエラーが発生している
   - ネットワークの問題

3. **レスポンスは返ってくるがUIに表示されない**
   - レスポンス形式の問題
   - 拡張機能の解析エラー
